# Educontabel

Um projeto educacional desenvolvido como um Pen no CodePen.

## Descrição

Educontabel é uma aplicação web interativa criada para fins educacionais. O projeto demonstra o uso de HTML, CSS e JavaScript para criar uma interface responsiva e dinâmica.

## Tecnologias

HTML, CSS, JavaScript, TypeBot

## Origem

Este projeto foi originalmente criado no CodePen por **skytzen**.

**Original URL:** [https://codepen.io/skytzen/pen/gbLLaPB](https://codepen.io/skytzen/pen/gbLLaPB)

## Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/EuBluezinn/Educontabel.git
```

2. Abra o arquivo `index.html` em seu navegador
